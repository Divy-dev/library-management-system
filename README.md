# Library Management System

##  Description
This is a Java console-based application to manage library records.

##  Features
- Add book
- View books
- Delete book

##  Technologies Used
- Java
- OOP Concepts
- ArrayList

##  How to Run
1. Compile:
   javac LibraryManagementSystem.java

2. Run:
   java LibraryManagementSystem

##  Project Type
Console-based application
